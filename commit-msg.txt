v0.5.5: Add self-update command and version flag

## Features
- `lecoder-cgpu upgrade` - One-command update to latest version
- `lecoder-cgpu upgrade --check` - Check for updates without installing
- `lecoder-cgpu upgrade --beta` - Install beta version for testing
- `lecoder-cgpu --version` or `-v` - Display current version
